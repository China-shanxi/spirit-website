# 精神小伙&精神小妹潮流网站
纯前端静态土酷霓虹风网站，可部署GitHub Pages在线打开
## 网站板块
1. 霓虹自动轮播首页
2. 精神小伙/小妹人物图鉴卡片
3. 社会精神语录展示墙
4. 潮流穿搭相册分区
5. 家族留言板（本地存储留言，刷新不消失）
6. 背景音乐播放开关
## 特色
- 暗黑霓虹粉蓝渐变发光UI，氛围感拉满
- 手机、电脑自适应布局
- 平滑滚动导航，自动轮播横幅
- 留言本地存储，不会上传数据
## 部署教程
1. 新建GitHub仓库，上传 index.html / style.css / script.js / README.md
2. 仓库 Settings → Pages
3. 分支main，根目录部署，等待1分钟生成在线链接
## 自定义修改
1. 替换相册、人物头像：修改css内background色块为图片链接
2. 更换语录：修改word-item内部文字
3. 添加背景音乐：给audio标签src填入mp3链接
4. 修改文案、配色：调整css渐变颜色值
